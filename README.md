## 这是一个什么教程？
这是一个关于词法分析器生成器**Flex**与语法分析器生成器**Bison**的系统化教程，面向公众，开源、开放、免费。
本教程适合那些想要自制编程语言或者基于其它目的想要掌握Flex与Bison的人。  
想要流畅地阅读本教程，请保证你有C语言的基础以及基本的编程思想。
